Are we (I)DE yet?

An overview about the state of Rust development environments and their integrated bretheren.

This overview is so much beta, it's likely that I have forgotten many things!

Text Editors

  • Emacs

    For Emacs there is RustMode installable from source or via MELPA, it brings

    • syntax highlighting
    • snippets

    With Flycheck comes

    • syntax checking

    Racer gets you

    • code completion

  • Geany

    Without further trouble Geany brings

    • syntax highlighting
    • build scripts (Cargo support coming soon)

  • Kate

    Kate has included

    • syntax highlighting

    With some help of Racer you get

    • code completion

  • Sublime Text 2/3

    Rust Package brings

    • syntax highlighting
    • snippets
    • build scripts

    RustAutoComplete uses Racer to bring

    • code completion

  • Vim

    rust.vim offers

    • syntax highlighting

    With Racer you get

    • code completion
    • goto definition

Integrated Development Environments

  • Eclipse

    The plugin RustDT works best when Racer is installed and then offers

    • syntax highlighting
    • snippets
    • project wizard
    • goto definition
    • gdb debugging

  • Visual Studio

    With Visual Rust you get

    • syntax highlighting
    • project support
    • build scripts

  • SolidOak

    SolidOak is the first Rust-specific IDE. It has integrated

    • syntax highlighting
    • code completion

    It wraps Cargo instructions for creating, building, testing and running projects.

Meta

  • Racer

    Racer is a code completion tool for Rust that is developed standalone. It is the basis for code completion in pretty much every setup.
